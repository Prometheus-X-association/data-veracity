# DVA processing module (Python)

> [!IMPORTANT]
> This project is built with [`uv`](https://docs.astral.sh/uv/).
> If you are a developer, please [install `uv`](https://docs.astral.sh/uv/getting-started/installation/).


## How to run

```console
$ uv run dva-processing
```


## Run unit tests

```console
$ uv run pytest
```
